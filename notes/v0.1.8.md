# Codex Web Manager v0.1.8

本次版本重点提升移动端会话操作、实时状态可靠性、暗色界面一致性和浏览器安装体验。

## 移动端与会话管理

- 手机端支持右滑置顶、左滑删除；桌面端支持右键置顶、取消置顶和删除会话。
- 删除操作增加确认弹框，运行中的会话会阻止删除，避免误删正在执行的任务。
- 每个会话独立保存未发送草稿，切换会话后输入内容不会互相覆盖。
- 后台会话完成时不再自动跳转或替换当前正在查看的对话。
- 优化长会话加载和消息刷新，减少手机端进入会话时的等待与界面抖动。

## 通知与实时状态

- 新增站内任务通知中心，普通局域网 HTTP 环境也能查看任务完成提醒。
- 铃铛显示未读数量，支持打开对应会话、单条清除和清除全部。
- 在可信 HTTPS 环境下可额外启用浏览器系统通知。
- SSE 实时连接增加“实时、重连中、离线”状态提示。
- 修复停止后详情接口仍将会话判断为运行中，以及返回列表后状态重新变回运行中的问题。
- 强制中断状态会持久保存，只有检测到真正的新任务开始后才会清除。

## 界面与主题

- 重新设计暗色主题，统一导航、会话列表、对话区、输入框、账号页、通知弹框、Profile 弹框和首次设置页。
- 修复暗色背景下按钮文字、输入框和弹框出现白色反色块或对比度不足的问题。
- 新增跟随系统、浅色、深色三种主题模式，并在本机保存选择。
- 优化手机顶部按钮、发送按钮、运行状态框和底部导航布局。

## 安装与代理支持

- 增加 Web App Manifest、Service Worker 和 180/192/512 像素应用图标。
- Windows 可下载 `.url` 桌面快捷方式，目标地址使用浏览器当前实际访问 URL。
- 静态资源、API、SSE、manifest 和 Service Worker 支持反向代理路径前缀。
- Android Chrome 通过可信 HTTPS 地址访问时可安装为独立 PWA；普通 HTTP 仍可添加网页快捷入口。
- 手机安装说明会按 Android 或 iPhone 显示对应步骤，并在复制被浏览器阻止时提供明确反馈和可长按地址。

## 安装与升级

下载 `codex-web-manager-0.1.8-win-x64.zip`，解压后运行 `Start Codex Web Manager.cmd`。

从旧版本升级时可以直接替换程序文件；请保留本机 `.env` 和 `data` 目录。默认端口仍为 `6688`。
