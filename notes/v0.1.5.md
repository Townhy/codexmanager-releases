## 主要更新

- 网页执行器迁移到 Codex app-server，使用 `thread/resume` 和 `turn/start` 继续本地会话。
- 支持运行中追加引导（`turn/steer`）和精确停止（`turn/interrupt`）。
- 支持从网页强制中断 Codex Desktop 发起的任务，不关闭 Desktop 主窗口。
- 后台重启后可识别并停止遗留的 `codex exec resume` 进程。
- 优化手机端输入区：停止操作移到运行状态栏，发送箭头完整置于输入框内。
- 修复停止接口版本不一致、任务归属丢失和停止后状态仍显示“处理中”的问题。

## 安装

下载 `codex-web-manager-0.1.5-win-x64.zip`，解压后运行 `Start Codex Web Manager.cmd`。目标电脑需要 Node.js 22.5+ 和 Codex Desktop。

> 从网页强制停止 Desktop 任务时，会同时中断同一 Profile 中正在执行的其他 Desktop turn。
