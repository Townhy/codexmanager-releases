## Codex Web Manager v0.1.3

本版本重点改善手机端使用体验、长会话加载速度和网页任务控制能力。

### 移动端与性能

- 首页会话数据由约 2.75 MB 缩减到约 39 KB。
- 长会话默认加载最近 300 条消息，支持继续加载更早记录。
- 加载历史消息时保持当前滚动位置。
- 历史消息与 Markdown 渲染增加缓存，输入框打字不再触发整个会话重绘。
- 优化首次 SQLite 会话扫描，限制超长预览字段读取。

### 任务控制

- 网页任务可按会话停止，不再依赖当前页面保存的 Job ID。
- 刷新页面、重新进入会话或换设备后仍可停止网页任务。
- 兼容旧后端的 Job ID 停止接口。
- 取消任务后不再错误显示为发送失败。
- 超过 5 分钟没有继续写入的残留运行状态显示为“已中断”，可直接继续会话。

### 会话权限

- 每个会话可选择“跟随 Profile”“工作区可写”或“完全访问”。
- 权限选择只影响网页发起的下一轮任务，并按会话保存在浏览器中。
- 服务端严格校验权限枚举，避免注入任意 CLI 参数。

### 实时更新

- 前端构建版本通过 SSE 推送。
- 前端文件更新或后端重启后，浏览器会自动检测新版本并刷新。
- 会话状态继续通过 SSE 实时同步。

### 安全提示

“完全访问”会绕过 Codex 审批和沙箱限制，只应在可信会话和可信设备上使用。
