# Codex Web Manager v0.1.10

本版本集中完善移动端远程对话、Codex App Server 集成、任务恢复和宽屏界面。源码与发布包继续排除本机账号、API Key、Token、Profile 数据、会话文件、日志和 `.env`。

## Codex 对话能力

- 增加模型推理强度选择：自动、最小、低、中、高、极高，并通过 `turn/start.effort` 传给 Codex App Server。
- 输入框改为 Codex 风格的 `+` 菜单，统一选择照片、文件、文件夹和当前 Profile/项目中已安装的 Skills。
- 增加工具执行过程折叠展示，覆盖命令执行、文件修改、MCP/动态工具调用和退出状态。
- 接收 `turn/diff/updated`，可在对话中展开查看本轮文件修改结果。
- 增加移动端审批模式，网页启动的轮次可在命令或文件修改前选择允许一次、本轮允许或拒绝。
- 从 Codex rollout 提取加粗的简短执行标题，Desktop 任务也可显示类似客户端的当前步骤，不展示完整内部推理。

## 会话与同步

- Web Manager 重启后恢复观察仍在执行的网页任务。
- 修复恢复观察会话打开时报 `Invalid time value` 的问题。
- 网页轮次完成后刷新 `session_index.jsonl`，改善 Codex Desktop 重启后的会话索引同步。
- 会话先返回最近 80 条消息，再后台补充到 300 条，降低手机首次打开等待时间。
- SSE 重连后自动刷新会话列表和当前对话，减少网络波动后的状态滞后。
- 记录 App Server 初始化能力，便于适配不同 Codex 版本的协议变化。

## 阅读与交互

- 长代码块支持折叠、展开和复制，手机查看代码更方便。
- 手机输入区的权限、审批、模型、推理强度和发送按钮改为单行平均分布。
- 收紧手机端段落、列表项和消息间距，减少小屏幕无效留白。
- 深色主题下模型和推理强度选择器与全局配色统一，不再出现白底控件。
- 宽屏对话正文扩大到约 1040px，减少大屏右侧空白。
- Profile 运行项增加明确的“重启”按钮，可重启对应 Codex Desktop 实例。

## 安装升级

下载 `codex-web-manager-0.1.10-win-x64.zip`，解压后运行 `Start Codex Web Manager.cmd`。默认端口仍为 `6688`。
